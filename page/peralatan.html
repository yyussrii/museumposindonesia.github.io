RUANG KOLEKSI PERALATAN

Pos Indonesia dalam melayani para pelanggannya tidak terlepas dari peralatan dan perlengkapan yang dipergunakan, peralatan-peralatan baik yang masih dipergunakan atau yang sudah tidak dipergunakan lagi di display di dalam ruang penyajian Museum Pos Indonesia,

Gerobak Angkut Pos (1870)

Salah satu peralatan yang dipergunakan adalah Gerobak Angkut Pos yg pernah dipergunakan di Kantor Pos Ambon tahun 1870 Perlatan tersebut dipergunakan untuk mengangkut kiriman-kiriman pos dari Stasiun Kereta Api untuk selanjutnya setelah dipilah / dilakukan penyortiran dikirimkan ke alamat tujuan

Bis Surat

Bis surat merupakan sarana pelayanan untuk memudahkan masyarakat dalam mengeposkan kirimannya. Bis surat biasanya ditempatkan di tempat-tempat yang strategis di pinggir jalan, sehingga pengirim tidak perlu ke kantor pos, hanya perlu memasukkan suratnya ke bis surat terdekat, kiriman yang dapat melalui bis surat adalah surat, warkatpos,kartu pos setelah terlebih dahulu dibubuhi prangko. Sejak pertama kali digunakan tahun 1829 di kantor Batavia, bentuk dan ukuran bis surat mengalami beberapa perubahan. Bis surat pertama untuk umum disediakan di kantor pos semarang tahun 1850 dan kantor pos surabaya tahun 1864.

Sepeda Pos

Alat pengangkut roda dua, berupa sepeda Merek Falter, yang di buat di jerman barat pada tahun 1938 yang dipergunakan sekitar tahun 1900 di Pos Indonesia, untuk mengangkut barang-barang pos dari stasiun kereta api.

Tas Pengiriman Surat

Tas yang di gunakan untuk menyimpan Kiriman Surat oleh pengantar Pos Pada Zaman Hindia Belanda Tahun 1930. Biasanya dipakai oleh pengantar pos menggunakan sepeda pos.

Topi atau Pet

Topi atau Pet yang pernah di pergunakan oleh para pegawai di kantor pos

Mesin Hitung Elektronik

Burroughs Visible Adding Machine Class 3 Merupakan alat buatan William Burroughs yang dibuat pada tahun 1925 di Detroit, Amerika Serikat. Burroughs Class 3 Merupakan seri kelanjutan dari model sebelumnya yaitu Burroughs Visible Adding Machine Class 1. Burroughs Class 3 Berfungsi Sebeagai Mesin Hitung yang dapat menambahkan angka dan memiliki mekanisme pencetakan. Mesin Hitung elektronik ini dipergunakan untuk menghitung uang kas di kantor pos yang digunakan pada jaman PTT (Pos Telegrap dan Telepon)

Kereta api Kantor pos

Tahun 1871 dibuka kereta api pertama antara Weltervreden (Gambir) dan Batavia (Jakarta Kota). Angkutan kereta api digunakan untuk angkutan pos. Pada tahun 1878 dibuat peraturan untuk pengangkutan pos sepanjang jalan pos rute Anyer - Bandung, Yogyakarta - Surabaya "Sporwegpostkantoren" atau Kantor Pos dengan pembukaan Kereta Api, dengan dibukanya jalan kereta api, surat dari Batavia ke Surabaya hanya perlu waktu satu setengah hari.

Untuk perjalanan ditempat KA Ekspress tidak berhenti, perangkat penangkap pos dipasang, yang terdiri dari tiang tempat tergantung kantong pos dipasang disebut "vanginrinchting" , dengan cara ini Kereta Api dapat mengambil/menyambar kantong pos tanpa berhenti. Sedangkan untuk tempat yang tidak ada kantor pos, pembantu diberikan kesempatan untuk membeli barang di Benda pos di kantor pos kereta api yang juga melayani pencatatan surat.

Tahun 1892 dibuka layanan Pos Ambulans, yaitu pengangkutan kiriman barang pos dipercayakan kepada konduktor Kereta Api. Konduktor ini memiliki sortir rak ditempat-tempat pemberhentian dimana tidak terdapat kantor pos, konduktor dan kepala stasiun saling menukarkan kiriman pos. sementara penjualan pos dilakukan di stasiun dan menerima upah 1 gulden setiap bulan.

Untuk meningkatkan pengangkutan kiriman pos, semua perusahaan Perkereta-apian milik negara maupun swasta memiliki kewajiban untuk melakukan angkutan kiriman pos tertutup di bawah pengawasan petugas Kereta Api atau Trem. Pada waktu itu banyak Kereta Api dilengkapi dengan bis surat untuk melayani penumpang yang sedang bepergian
